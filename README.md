# ecommerce-customer-segmentation
ML-based customer segmentation system using Decision Tree to classify users and send personalized notifications in e-commerce.
# 🛒 E-commerce Customer Segmentation & Notification System

# Project Overview

This project uses Machine Learning to classify e-commerce customers into different categories based on their behavior and provides personalized notifications.

## 🎯 Objective

* Identify customer types
* Improve customer engagement
* Provide targeted offers and recommendations

## 🧠 Model Used

* Decision Tree Classifier

## 📊 Features

* Total Orders
* Total Spending
* Last Login Days

## 🧩 Customer Categories

* 🟢 Regular Customers
* 🟡 Average Customers
* 🔴 Low Engagement Customers

## 🔔 Notification System

* Regular → Exclusive discounts
* Average → Coupons and offers
* Low → New arrivals and promotions

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Pickle

## 🚀 How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Train model:

```
python src/train_model.py
```

## 📦 Output

* Trained model saved as `model.pkl`

## 💡 Use Case

This system can be used by e-commerce platforms to improve marketing strategies and customer retention.
